Example
=======

To quickly get started with ATACofthesnake, functionality to download and play around with example data is provided.
This data is publically available, and was generated by Gaidatzis et al. (2025) - Cele dataset, 10.1101/2025.09.04.674194 and Yoshide et al. (2019) - Immgen dataset, 10.1016/j.cell.2018.12.036.
The first dataset contains time series data in C. elegans, the second dataset contains data from the developing T-cell lineage in mice.
To download the data, the following command can be used:

.. code-block:: bash

    ATAC_example -h

By default both datasets will be downloaded. In case you are interested in only one, a positional argument can be used:

.. code-block:: bash

    ATAC_example cele

or 

.. code-block:: bash

    ATAC_example immgen

An output directory can be specified by using `-o` or `--outputdir`. By default, the data will be downloaded a subdirectory 'aos_example' in the current working directory.
